What is Noch
Noch is a school project that was originally built using meteorjs, nodejs, mysql and basic html/css/javascript. It was later adapted into ASP.NET MVC using signalR and SqlServer. This version of Noch is built with Meteor but using Angular2 rather than the builtin Blaze-html-templates package. It will also use MongoDb as the database server... maybe.
